## 2. The Problem

### SNS 의 시대가 끝나가고 있다.

| 플랫폼 | 현재 정체 |
|---|---|
| **Instagram** | 광고판이 됐다. 인플루언서의 협찬과 무한 비교. |
| **Facebook** | 노년의 공간으로 밀려났다. |
| **KakaoTalk** | 업무 도구가 됐다. "친구야 잘 지내?" 가 사라졌다. |

### MZ세대는 이미 거부하고 있다.

- 카톡 프로필 사진 안 바꾸기
- 인스타 스토리 닫기
- 친구 추가 망설이기

**→ 단순한 트렌드가 아니라, 디지털 시대에 살아남기 위한 자기보호.**

### 그리고 더 큰 사회 문제

2026년 한국에서 매일 듣는 뉴스:
- **학생 폰 중독** — 부모와의 싸움, 폭력적 변화
- 가장 큰 원인 = SNS 의 **비교, 박탈, 무한 스크롤, 도파민 알림, 자랑 경쟁**
- 사춘기의 약한 자아가 견디기엔 너무 가혹한 환경

### 그러나 사람들은 여전히 연결을 원한다.

**다만 다른 방식의 연결을.**

## 3. The Solution — Maslow 5단계 SNS

### 기존 SNS 는 3단계에서 멈춘다.

대부분의 SNS 는 매슬로우 피라미드의 **3단계 (소속/애정)** 에서 정체. 친구를 만들고 좋아요를 받고 댓글로 연결되지만, 한계는 명확하다 — **비교, 박탈감, 피상적 연결.**

### Do : 하다 는 5단계 (자아실현) 에 답한다.

| 단계 | 욕구 | 기존 SNS | Do : 하다 |
|:-:|---|---|---|
| 1 | 생리적 | 없음 | 건강 챌린지 (수면 · 식사 · 운동) |
| 2 | 안전 | 없음 | 금연 · 금주 · 재테크 |
| 3 | 소속·애정 | 친구·좋아요 (피상적) | 도전 동료 (목적 기반 · 신청 X) |
| 4 | 존중·인정 | 좋아요 수 (비교 압박) | 4가지 응원·평가 · 작은 동료 단위 |
| 5 | **자아실현** | **거의 없음 ★** | **100일 완주 · 박제 · "더 나은 나"** |

> Instagram: "내가 얼마나 예쁜지 보세요" *(3단계 피상적 인정)*
> KakaoTalk: "친구야 잘 지내?" *(3단계 피상적 소속)*
> **Do : 하다 : "오늘도 한 걸음 나아갔어요"** *(5단계 자아실현)*

## 4. Why Now?

### 시장 신호 ① — MZ세대의 명시적 거부

기존 SNS 의존도 감소 (스토리 닫기, 친구 추가 망설임) — **다른 방식 연결을 원하는 명백한 신호.**

### 시장 신호 ② — 사회 문제 직결

학생 폰 중독 → 가족 폭력 → 사회 비용. **반대로 만든 SNS** 가 정책·교육 기관·학부모 시장의 자연 우호 세력.

### 시장 신호 ③ — 매슬로우 5단계 솔루션 부재

"자기계발 앱" (혼자 사용) 과 "SNS" (피상적 연결) 사이의 비어있는 영역 — **소셜 + 자아실현 결합** 의 첫 정의자.

### 기술 신호 — 1인 개발 운영 가능 시기

- Supabase / Expo / Cloudflare R2 / Expo Push 등 인프라 성숙
- LLM 기반 1인 바이브 코딩 — Phase 1 베타를 **3개월 1인** 으로 완성

### 홈 화면 — 도전 인연들의 하루 (v2.5)

> **사상 전환**: v2.1~v2.4 의 "X 빼기" (비교·친구신청·알림·무한스크롤) 만으론 챌린지 도구가 됐다.
> v2.5 = **버릴 건 망가진 방식, 지킬 건 욕구 자체**.

| 인간 욕구 | 망가진 방식 (버림) | Do : 하다 새 방식 (살림) |
|---|---|---|
| 인정받기 | 좋아요 수 비교 | 목격받기 (동료가 내 여정을 지켜봄) |
| 소속 | 팔로우·친구신청 | 도전 인연 (목적 기반 자동 연결) |
| 타인과 이어짐 | 도파민 피드 | 되어가는 과정 피드 |

홈 = 내 대시보드 아닌 **도전 인연들의 하루** 피드. 같은 앱 안의 4가지 챌린지 종류 + 관심 도전이 한 피드에 공존:

```
🤫 혼자만의 다짐 — 사적 일기 (조용한 자기와의 약속)
🙋 응원받는 도전 — 도전자 무대 + 응원자 객석 (지인 응원 받기)
🤝 함께 도전     — 친밀 모임 (작은 단위 친밀함)
🌍 누구나 합류    — 광장 / 공익 / 사회공헌
✨ 관심 도전      — 사용자 명시 의도로 발견
```

각 분류별로 카피·UI·알림·박제·인연 모두 다르게 — 시스템적 일관성.

### 4가지 방 종류

| | | 도전자 | 응원자 | 페르소나 |
|:-:|---|---|---|---|
| 🤫 | **혼자만의 다짐** | 본인 | 본인 | 조용한 자기 기록 |
| 🙋 | **응원받기** ⭐ | 본인 1명 | 초대 지인 N명 | 워킹맘 + 가족 응원 / 100일 다이어트 선언 |
| 🤝 | **함께 도전하기** | 멤버 다수 | 멤버 다수 | 토익 스터디 / 운동 메이트 |
| 🌍 | **누구나 합류** | 멤버 다수 | 멤버 다수 | 명사 / 공익 / 사회공헌 |

### 7가지 차별화 약속

1. **비교 압박 X** — 큰 숫자 없음 (100+ → "99+"), 인증률 순위 X
2. **친구 신청 X** — 챌린지 참여로 자동 동료
3. **조용한 알림 4원칙** — 도파민 루프 차단
4. **끝이 있는 도전** — 무한 스크롤 X, 7/30/100/365일 기간제
5. **작은 동료 단위** — 3-5명 친밀 단위 default
6. **의미 있는 평가** — 좋아요 1차원 X, 8가지 의미 분리
7. **더 나은 세상** — 사회공헌 카테고리 (환경·기부·봉사·공익)

## 8. Tech Stack & Architecture

|---|---|

| **Landing** | GitHub Pages (정적, 무료) |

## 12. Closing — Why This Will Work

### 한 문장

> **"내가 도전하는데 내 지인들이 응원해줘요.**
> **앞으로 100일 동안 10kg 줄일 거야.**
> **나를 응원해주고 지지해줘."**

이 한 문장이 **응원받기 방** 의 원초적 시나리오. 인스타·카톡 어디에서도 해결되지 않는 본질적 욕구.

### 정직한 약속

- **30명 중 10명이 추천 안 하면 멈춥니다.**
- 인터뷰 결과 모든 항목 공개.
- 베타 후 의사결정 = 사용자 인터뷰 데이터 기반.

### 이 카테고리의 첫 정의자가 되는 일

비교 없는 응원. 친구 신청 없는 동료. 조용한 알림. 끝이 있는 도전. 박제 자산화.

기존 SNS 의 반대편에서, **5단계 자아실현 SNS** 라는 새 카테고리를 정의합니다.
