## 1. 절대 규칙 (AI 티 제거)

- 시스템 픽토그램에 이모지 금지 → 통일된 **라인 아이콘**(lucide, stroke 1.6~1.8).
- **이모지 예외는 딱 2곳뿐:**
  - **응원 4종**(인증 탭 반응): 🔥 👏 💪 ❤️
  - **평가 4종**(하다 구경 카드): ✨기발 / 😱대단 / 🥹뭉클 / 💫새로움
  - 그 외(탭바·뱃지·상태·빈 상태)는 전부 라인 아이콘.
- 흰 카드 + 그림자 남발 금지 → 색 면으로 구분. 그림자는 화면당 1곳(앵커/FAB).
- 색 신호등 금지(오렌지+초록버튼+💚+🟢 혼재) → 아래 세만틱 1체계로 통일.
- 한 화면 강조(앵커)는 1개. 나머지는 조용히.

## 2. 컬러 토큰

bg #FAF7F2 / surface #FFFFFF / line #ECE6DE
brand #FF6B35 / brandPress #E85A28 / brandTint #FFF1E9
tintWarm #FFF1E9 (응원·오늘의 나) / tintCream #FBF3E4 (완주·박제) / tintSage #EDF1EC (합류·발견)
**done #5C8A6A / doneTint #EAF1EC ← 완료/성공 세만틱. 기존 초록버튼·💚 전부 이걸로 대체**
ink #2A2622 / sub #6B645C / faint #9B938A / onBrand #FFFFFF

- 면색(tint*)은 한 화면 최대 2종.
- 오렌지 = 앵커 액션 + 활성 탭 + D-day 에만.
- 세이지(done) = 완료 체크 · "오늘 인증 완료" 버튼 · 성공 상태.

## 3. 타이포 (Pretendard)

## 4. 간격·모서리
space 4/8/12/16/20/24/32 · radius sm12 md16 lg20 pill999 · 좌우 여백 20 고정 · 섹션 세로 28~32.

## 5. 그림자
거의 안 씀. 앵커 카드/FAB만. 일반 카드는 면색+1px line으로 구분.

## 6. 화면 위계 (홈)
오늘 할 일(앵커, tintWarm+오렌지 CTA) → 오늘 완료(세이지 체크, 압축) → 끝낸 하다(가로 칩) → 인연들의 하루(라인 리스트) → 응원(보조) → 합류(sage). 위→아래로 무게 감소. 평면 나열 금지.

## 7. 응원/평가 컴포넌트 (이모지 예외)
- 응원 pill(인증 탭): 높이 42, 이모지 19px. 선택=brand 테두리+brandTint+카운트(brand). 미선택 회색.
- 평가 박스(하다 구경): 4칸 가로. 이모지 24px + 라벨 + 카운트. 선택=brand 테두리+brandTint.
- 이모지 렌더링: font-family에 'Apple Color Emoji','Segoe UI Emoji','Noto Color Emoji' fallback 필수.

## 8. 이모지 → 라인 아이콘 매핑
탭바 홈/내하다(flag)/기록(film)/해냈어요(trophy) · 방탭 대화(chat)/인증(camera)/기록(film)/현황(bar-chart)/박제(trophy) · 방종류 누구나(globe)/응원받기(heart)/다함께(handshake) · streak(flame) · 종료(flag) · 알림(bell) · 둘러보기(compass).
챌린지 썸네일은 아이콘보다 사용자 인증 사진 우선.

## 9. 문구 톤 (비교 거부 — 유지)
"줄세우지 않고 서로에게 용기를", "조용한 알림 원칙—숫자 대신 점 하나", "N명이 용기를 얻었어요". 랭킹/1등/더 많이 금지.

## 10. Claude Code 적용 순서
1. theme/colors·type·spacing 토큰 생성/교체.
2. 공통 컴포넌트(SectionCard/ListRow/PrimaryButton/Chip/응원pill/평가box) 토큰 기반 작성.
3. 홈부터 6번 위계대로 리팩터(AS-IS→TO-BE 설명 후 실행).
4. 전 화면 이모지 grep → 라인 아이콘 교체(응원 4종·평가 4종은 유지).
5. 초록 CTA·💚·🟢 → done 계열로 교체.
이 문서에 없는 색·폰트·간격은 새로 만들지 않는다.

# === v2 추가 (2026-06-22 세션) — 아래가 최신, 충돌 시 우선 ===

## 11. 헤더 구조 (전 화면 통일)
- **로고 = icon.png 심볼 그대로** (점·원·괄호 = 나·지인·누구나). 텍스트 괄호 `(⊙)` 금지. 헤더용은 배경 투명 SVG로 export.
- 메인 탭 헤더: **좌측 = 프로필**(아바타+닉네임+한 줄), **우측 = 벨 + 햄버거(≡)** 2개.
  - 프로필 탭 → 내 프로필. 햄버거 탭 → 설정.
- 방·상세 헤더: 좌측 뒤로가기 + 타이틀, 우측 액션(초대 등).
- 로고는 **홈 화면 맨 아래 마감 문구 밑에 워터마크 서명**으로 1회 (opacity ~0.5). 홈 복귀는 탭바 홈이 보장.

## 12. 탭바 + 구경 동선
- 탭바 5칸 유지: **홈 · 내 하다 · (+) · 기록 · 해냈어요**.
- **하다 구경 진입점은 홈 최상단 얇은 바 1곳으로 통일.** (기존 홈 하단·내 하다 하단 진입점 전부 제거.)
- 구경 바는 앵커보다 가볍게(회색 텍스트, 작은 높이). 콜드스타트에선 크게.
- **관심 분야 추천은 구경으로 통합** — 구경 상단에 "내 관심 분야 우선" 영역 + 분야 필터 칩. 홈에 별도 관심추천 섹션 두지 않음.

## 13. 분야 아이콘 표 (lucide, 전 화면 고정)
어느 화면이든 같은 분야는 같은 아이콘. (Tabler명 → lucide 동일/유사명)
| 분야 | 아이콘 |
|---|---|
| 건강 | heartbeat (활동/맥박) |
| 운동 | run / footprints |
| 학습 | book |
| 창작 | palette |
| 자기계발 | briefcase |
| 재테크 | coin / coins |
| 라이프 | globe (world) |
| 관계 | users |
| 사회공헌 | heart-handshake |
| 기타 | dots / more-horizontal |
> 주의: 건강 하다(예: 탄수화물 없는 한끼)에 hand-heart/하트를 쓰지 말 것 — 하트는 응원 의미와 충돌. 분야 아이콘은 위 표만.

## 14. 화면별 확정 사항 (v2)
- **홈 위계** (6번 재확인 + 구경 바 추가): 구경 바(얇게) → 오늘 할 일(앵커) → 오늘 완료(세이지 압축) → 끝낸 하다(가로 칩) → 인연들의 하루(라인 리스트) → 응원(보조) → 합류(sage) → 마감+로고 서명.
- **내 하다**: 진행 카드는 연속일수를 큰 숫자(26px)로, 진행률 바 두껍게, 상태 배지(완료=세이지 tag / 인증 전=코랄 tag). 응원하는 하다=압축 행, 끝낸 하다=크림 리스트, 지난 하다=접힘.
- **만들기 분야선택**: 10개 분야 **2열 그리드 + 가로 배치**(아이콘 좌 + 라벨 우, 카드 높이 낮게). 선택 1개만 brandTint+오렌지 테두리.
- **내 프로필**: 아바타 카메라 뱃지, 관심분야 칩(분야 아이콘), "나의 발자취" 3칸(완주/최고연속/받은응원 — 줄세우기 아님), 완주보관함·다짐내역·한잔내역은 **좌측정렬 리스트 행**(아이콘+라벨+chevron).
- **설정**(햄버거): 알림 토글 그룹 + 계정 그룹(문의·약관·로그아웃) 분리. 계정 삭제만 빨강 텍스트.
- **방 인증**: 응원 4종 pill(미선택 회색→선택 brand). "오늘 인증 완료" 버튼 = done(세이지), 형광 초록 금지.
- **구경**: 분야 필터 칩 + 관심 분야 우선 + 평가 4종 박스(선택 시 brandTint).
