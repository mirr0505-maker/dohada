시행일자: 2026년 6월 22일 · 베타 v1.0.0 · 앱: Do : 하다 (app.dohada.beta) · 운영자: 김기두

Do : 하다(이하 "서비스") 이용자는 언제든지 본인 계정과 관련 개인정보의 삭제를 요청할 수 있습니다. 본 안내는 계정 삭제 방법과, 삭제되는 정보·보존되는 정보·보관 기간을 설명합니다.

1. 계정을 삭제하는 방법

방법 A — 앱 안에서 직접 (권장)

  1. 앱 실행 후 우상단 프로필(MY) → 내정보 화면으로 이동합니다.
  2. 화면 하단의 로그아웃 아래 "계정 삭제"(빨간색)를 누릅니다.
  3. 삭제 영향 안내를 확인하고 동의 체크박스를 선택한 뒤 다음으로 넘어갑니다.
  4. 최종 확인을 누르면 계정이 즉시 삭제(비활성화·익명화)됩니다.

방법 B — 이메일 요청

앱에 접속하기 어려운 경우, mirr0505@gmail.com 으로 가입에 사용한 계정 정보와 함께 삭제를 요청해 주시면 지체 없이 처리합니다.

계정 삭제는 유예 기간 없이 즉시 처리됩니다. 삭제 후에는 같은 계정으로 다시 로그인할 수 없습니다 (다른 계정으로는 새로 시작하실 수 있습니다).

2. 삭제·익명화되는 데이터 (즉시)

  • 계정 식별 정보: 이메일, 닉네임, 프로필 이미지 — 삭제 또는 익명화(닉네임은 "탈퇴한 사람"으로 대체)
  • 본인인증 정보: 생년월일, 휴대폰 번호 (수집된 경우)
  • 알림 정보: 푸시 토큰, 알림 설정값
  • 관심 분야 등 개인 설정값
  • 진행 중인 도전에서 자동으로 빠져나오며, 같은 계정의 재로그인은 영구 차단됩니다.

3. 보존되는 데이터 (익명화)

Do : 하다는 함께 도전한 동료들의 기록(박제)을 보호하기 위해, 여러 사람이 공유하는 공간에 남긴 콘텐츠는 작성자 정보를 익명화하여 보존합니다. 삭제한 이용자의 신원은 더 이상 식별되지 않습니다.

  • 공유 콘텐츠: 인증, 댓글, 기록, 대화, 완주 이야기, 다짐, 평가 — 작성자 익명화 후 보존
  • 결제 주문 내역: 응원 한잔·내기 등 주문 기록 — 관련 법령상 보관 의무에 따라 보존

4. 데이터 보관 기간

구분 보관 기간
개인을 식별할 수 있는 정보(이메일·닉네임·프로필·본인인증·푸시토큰 등) 삭제 요청 시 즉시 파기 (유예 없음)
익명화된 공유 콘텐츠(인증·댓글·기록·완주 이야기 등) 동료의 기록 보호를 위해 익명 상태로 보존
결제 주문 내역 관련 법령이 정한 법정 보관 기간
