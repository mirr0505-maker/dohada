왜 지금

기존 SNS가
변화해야 하는 시점에 와 있어요.

인스타그램은 광고판이 되었고, 페이스북은 젊은이의 기피 공간이 되었고, 카카오톡은 업무 도구가 되었어요. "친구의 일상" 은 알고리즘 깊은 곳에 묻혔고, 우리가 보는 것은 인플루언서의 협찬, 광고, 그리고 끝없이 스크롤되는 비교의 대상이죠.

MZ세대는 이미 이 흐름을 거부하고 있어요. 카톡 업데이트를 거부하는 것, 인스타 스토리를 닫는 것, 친구 추가를 망설이는 것 — 단순한 트렌드가 아니라 디지털 시대에 살아남기 위한 자기보호예요.

그리고 학생 폰 중독, 가족 폭력. 어린 학생들이 핸드폰을 놓지 못해 부모와 싸우고 폭력적으로 변한다는 뉴스. 가장 큰 원인 중 하나가 SNS 예요. 비교, 박탈, 무한 스크롤, 도파민 알림, 자랑 경쟁 — 사춘기의 약한 자아가 견디기엔 너무 가혹한 환경.

우리는 그 반대로 만들어요.

차세대 SNS의

7가지 약속

기존 SNS 와 무엇이 다른지, 정확히 풀어 적어요.

01

비교 압박 X

✕ 좋아요·평가의 큰 숫자 노출
✓ 0–99 만 그대로, 100+ 는 "99+"로 통일
✕ 멤버별 인증률 순위
✓ 가입 순 정렬 + 본인 위 (시간의 흐름 톤)
02

친구 신청 X

✕ 친구 / 팔로우 / 친구 추가 단어 자체 안 씀
✓ 챌린지 참여로 자동 동료 발생 — "도전 인연"
03

조용한 알림 — 가장 본질적 차별

학생들이 핸드폰 못 놓는 가장 큰 메커니즘이 실시간 알림 도파민 루프예요. 우리는 알림을 제공하되 4원칙으로 묶어요.
① 묶음 — 응원·좋아요는 1시간 단위로 묶어 "지난 1시간 동료 3명이 응원했어요" 한 건
② 즉시는 대화만 — 채팅·댓글만 즉시, 단 무음 + 진동
③ 조용한 시간 — 밤 10시 ~ 아침 8시 자동 보류
④ 일별 상한 — 사용자당 하루 최대 5건
04

끝이 있는 도전

✕ 무한 스크롤 / 무한 피드
✓ 기간 3 / 7 / 30 / 100 / 180 / 365일 챌린지
✓ 매일 1번 인증 후엔 라벨이 "오늘 인증 완료 · 내일 또 만나요"
05

작은 동료 단위

✕ "12,847 명이 함께 도전 중" 식 거대한 숫자
✓ "내 동료 5명 중 3명 ✓"
✓ 챌린지방은 평균 3–5명 친밀 단위가 기본
06

의미 있는 평가 4가지

✕ 좋아요 1차원 도파민
✓ 두 가지 평가 시스템 (의미 분리):
· 인증 응원 (방 안): 🔥 / 👏 / 💪 / ❤️ (감정 표현)
· 챌린지 평가 (둘러보기): ✨ 기발 / 😱 대단 / 🥹 뭉클 / 💫 새로움 (의미 평가)
07

더 나은 세상

✓ 사회공헌 카테고리 (환경 · 기부 · 봉사 · 공익)
✓ 둘러보기에 초록 뱃지 + 💚 "함께 만든 변화" 배너 + [+ 동참] 버튼
✓ 내가 나아질수록 세상도 함께 나아짐 — 슬로건의 실제 구현
