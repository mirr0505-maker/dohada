## [0] 공통 전제 (첫 지시에 같이 붙이기)

```
두하다 앱 디자인을 DESIGN_GUIDE.md 기준으로 리팩터한다. 시작 전 전제:
- 목업(design_final.html)은 Tabler 아이콘으로 그렸지만 앱은 lucide를 쓴다. 아이콘은 이름만 대응시켜 lucide로 쓰고, Tabler를 새로 설치하지 마라.
- 색·간격은 목업 눈대중이 아니라 DESIGN_GUIDE 토큰을 정본으로 쓴다.
- 이모지 예외는 응원 4종(🔥👏💪❤️)·평가 4종(✨😱🥹💫)뿐. 나머지 시스템 픽토그램은 전부 lucide 라인 아이콘.
- 큰 변경은 AS-IS → TO-BE를 먼저 설명하고, 내 확인을 받은 뒤 실행한다.
- 요청하지 않은 파일 추가나 범위 확장은 하지 마라.
이번 턴에는 코드를 고치지 말고, DESIGN_GUIDE를 읽고 적용 계획만 요약해줘.
```

## [1] 토큰

```
1단계: 토큰만 만든다. 화면 코드는 건드리지 마라.
- DESIGN_GUIDE 2번(컬러)·3번(타이포)·4번(간격/모서리)을 theme의 colors/typography/spacing으로 생성하거나 기존 값을 교체.
- DESIGN_GUIDE 13번 분야 아이콘 표를 분야→lucide 아이콘 매핑 상수로 만들어라(건강=heart-pulse, 운동=footprints, 학습=book, 창작=palette, 자기계발=briefcase, 재테크=coins, 라이프=globe, 관계=users, 사회공헌=heart-handshake, 기타=more-horizontal — lucide에 맞는 이름으로).
완료되면 무엇을 추가/교체했는지 목록으로 보여줘.
```

## [2] 공통 컴포넌트

```
2단계: 1단계 토큰으로 공통 컴포넌트를 만든다. 화면 적용은 아직 하지 마라.
- 헤더(좌측 프로필=아바타+닉네임, 우측 벨+햄버거 ≡)
- 탭바(홈·내하다·+·기록·해냈어요)
- 분야 아이콘 컴포넌트(매핑 사용)
- 상태 배지(완료=세이지 tag / 인증 전=코랄 tag)
- 응원 pill(미선택 회색 → 선택 brand 테두리+brandTint+카운트)
- 평가 box(4칸, 선택 시 brand)
- ListRow(좌측 아이콘+라벨+우측 chevron)
각 컴포넌트는 design_final.html의 해당 부분을 시각 기준으로 삼되 토큰을 쓴다. 완료되면 컴포넌트 목록을 보여줘.
```

## [3] 화면 — 홈부터 하나씩

```
3단계: 홈 화면을 리팩터한다. design_final.html의 홈, DESIGN_GUIDE 6번+14번 위계를 따른다.
위계 순서: 구경 얇은 바 → 오늘 할 일(앵커, tintWarm+오렌지 CTA) → 오늘 완료(세이지 압축) → 끝낸 하다(가로 칩) → 인연들의 하루 → 응원(보조) → 합류(sage) → 마감+로고 서명.
- 하다 구경 진입점은 홈 최상단 얇은 바 1곳뿐(기존 홈하단·내하다하단 진입점 제거).
- 로고는 맨 아래 워터마크 서명으로.
먼저 현재 홈의 AS-IS와 바꿀 TO-BE를 설명하고, 내 확인을 받은 뒤 실행해라.
```

이후 같은 틀로 한 화면씩 (한 번에 하나만):

```
다음 화면: [내 하다 / 만들기 분야선택 / 내 프로필 / 설정 / 방 인증 / 구경 / 기록 / 해냈어요 / 알림 / 콜드스타트 중 하나]
design_final.html과 DESIGN_GUIDE 14번을 기준으로, AS-IS→TO-BE 설명 후 실행. 한 번에 한 화면만.
```

---

## 화면별 핵심 체크 (3단계에서 참고)

| 화면 | 핵심 |
|---|---|
| 홈 | 위계 순서 + 구경 바 1곳 + 로고 하단 서명 |
| 내 하다 | 연속일수 큰 숫자, 상태 배지, 응원·끝낸·지난 압축 |
| 만들기 분야선택 | 2열 그리드 + 가로 배치(아이콘 좌+라벨 우) |
| 내 프로필 | 발자취 3칸, 내역은 좌측정렬 리스트 행 |
| 설정 | 알림 토글 그룹 + 계정 그룹, 계정 삭제만 빨강 |
| 방 인증 | 응원 4종 pill, 완료 버튼=세이지(형광초록 금지) |
| 구경 | 분야 필터 + 관심분야 우선 + 평가 4종 |
| 전 화면 | 이모지→lucide(응원·평가 4종만 예외), 초록·💚·🟢→done(세이지) |
