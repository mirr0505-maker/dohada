## 2. 왜 지금, 이걸 만드는가

### 진단 — SNS 의 시대가 끝나가고 있다

인스타그램은 광고판이 되었고, 페이스북은 젊은이들이 떠난 공간이 되었고, 카카오톡은 업무 도구가 되었다. "친구의 일상" 은 알고리즘 깊은 곳에 묻혔고, 우리가 보는 것은 인플루언서의 협찬, 광고, 그리고 끝없이 스크롤되는 비교의 대상이다.

MZ세대는 이미 이 흐름을 거부하고 있다. 카톡 프로필 사진을 바꾸지 않는 것, 인스타 스토리를 닫는 것, 친구 추가를 망설이는 것 — 이것은 단순한 트렌드가 아니라 **디지털 시대에 살아남기 위한 자기보호** 다.

### 더 큰 그림 — 학생 폰 중독, 가족 폭력

2026년 한국에서 우리는 매일 듣는다 — 어린 학생들이 핸드폰을 놓지 못해 부모와 싸우고 폭력적으로 변한다는 뉴스. 가장 큰 원인 중 하나가 SNS 다. **비교, 박탈, 무한 스크롤, 도파민 알림, 자랑 경쟁** — 사춘기의 약한 자아가 견디기엔 너무 가혹한 환경.

우리는 그 반대로 만든다.

### 그러나 사람들은 여전히 연결을 원한다

다만 **다른 방식의 연결** 이다.

## 4. 매슬로우 5단계 — Do:하다 의 자리

대부분의 SNS 는 매슬로우 피라미드의 **3단계 (소속/애정)** 에서 멈춘다. 친구를 만들고 좋아요를 받고 댓글로 연결된다. 그러나 이 단계의 SNS 는 한계가 명확하다 — **비교, 박탈감, 그리고 피상적 연결**.

Do : 하다 는 **5단계 (자아실현)** 에 답한다.

| 단계 | 욕구 | 기존 SNS | Do : 하다 |
|:-:|---|---|---|
| 1 | 생리적 | 없음 | 건강 챌린지 (수면 / 식사 / 운동) |
| 2 | 안전 | 없음 | 금연 / 금주 / 재테크 |
| 3 | 소속·애정 | 친구·좋아요 (피상적) | 도전 동료 (목적 기반, 신청 X) |
| 4 | 존중·인정 | 좋아요 수 (비교 압박) | 4가지 응원·평가 · 작은 동료 단위 |
| 5 | 자아실현 | 거의 없음 ★ | **100일 완주 · 박제 · "더 나은 나"** |

> 인스타: "내가 얼마나 예쁜지 보세요" (3단계 — 피상적 인정)
> 카톡: "친구야 잘 지내?" (3단계 — 피상적 소속)
> **Do : 하다 : "오늘도 한 걸음 나아갔어요"** (5단계 — 자아실현)

매슬로우는 UI 에 명시적으로 표시하지 않는다. 온보딩 카피, 챌린지방 헤더의 "함께 만든 변화", 내정보 태그라인에 **자연스럽게 녹인다**.

## 5. 어떻게 다른가 — 망가진 방식은 버리고, 욕구는 살린다

> **사상 (v2.5)**: 기존 SNS 가 채우려던 욕구(인정·소속·연결) 는 인간 본성이라 버리면 안 된다.
> 버릴 건 그 욕구를 채우던 **망가진 방식** (좋아요 비교·팔로우 신청·도파민 피드) 뿐.

| 인간 욕구 | 망가진 방식 (버림) | Do : 하다 새 방식 (살림) |
|---|---|---|
| 인정받기 | 좋아요 수 비교 | 목격받기 (동료가 내 여정을 지켜봄) |
| 소속 | 팔로우·친구신청 | 도전 인연 (목적 기반 자동 연결) |
| 타인과 이어짐 | 도파민 피드 | 되어가는 과정 피드 |

### 5.1 비교 압박 X · 그러나 "목격받기" 는 산다

- ❌ 좋아요·평가의 큰 숫자 노출 → ✅ 0~99 만 그대로, **100+ 는 "99+"** 로 통일
- ❌ 멤버별 인증률 순위 → ✅ **가입 순 정렬** + 본인 위 (시간의 흐름 톤)
- ❌ "OO 님보다 N일 앞섭니다" 카피 → ✅ 비교 카피 자체 지양
- ✅ **살린 욕구**: 동료가 내 인증을 보고 응원해주는 "목격" — 비교 없이 인정 욕구를 채움

### 5.2 친구 신청 X · 그러나 "소속" 은 산다

- ❌ 친구 / 팔로우 / 친구 추가 단어 자체 사용 안 함
- ✅ 챌린지 참여로 **자동 동료 발생** — "도전 인연"
- ✅ "친구 신청 없이도 동료가 자연스럽게"
- ✅ **살린 욕구**: 같은 목적을 향하는 사람과의 소속감 — 신청·수락 없이 목적이 모은다

### 5.3 조용한 알림 (가장 본질적 차별)

학생들이 핸드폰 못 놓는 가장 큰 메커니즘이 **실시간 알림 도파민 루프**. 우리는 SNS 의 기본인 알림을 제공하되 **4원칙** 으로 묶는다.

| | 원칙 | 구현 |
|:-:|---|---|
| 1 | **묶음** | 응원·좋아요는 1시간 단위로 묶어 "지난 1시간 동료 3명이 응원했어요" 한 건 |
| 2 | **즉시는 대화만** | 채팅·댓글은 즉시 (대화 흐름) · 단 **무음 + 진동만** |
| 3 | **조용한 시간** | 밤 10시~아침 6시 (KST) 모든 알림 자동 보류 → 아침 6시 묶음 발송 |
| 4 | **일별 상한** | 사용자당 하루 최대 5건 (옵션 X — default 강제) |

내정보의 5 토글로 사용자가 끄기도 가능 (`채팅 / 댓글 / 응원 / 동료 인증·기록 / 매일 안부`).

### 5.4 끝이 있는 도전

- ❌ 무한 스크롤 / 무한 피드 → ✅ **기간 7/30/100/365일** 챌린지
- ❌ 영원히 머무는 SNS → ✅ 완주 후엔 박제, 다음 도전으로
- ✅ 매일 1번 인증 후엔 라벨이 **"오늘 인증 완료 · 내일 또 만나요"** — 강제 휴식의 시작

### 5.5 작은 동료 단위

- ❌ "12,847 명이 함께 도전 중" 식 거대한 숫자 → ✅ "내 동료 5명 중 3명 ✓"
- ✅ 챌린지방은 평균 3~5명 친밀 단위가 default
- ✅ 둘러보기는 보고 싶을 때만 진입 (홈 메인 X)

### 5.6 의미 있는 평가 4가지

- ❌ 좋아요 1차원 도파민 → ✅ 두 가지 평가 시스템 (의미 분리)
  - **인증 응원** (방 안에서): 🔥 / 👏 / 💪 / ❤️ (감정 표현)
  - **챌린지 평가** (둘러보기 카드): ✨ 기발 / 😱 대단 / 🥹 뭉클 / 💫 새로움 (의미 평가)

### 5.7 더 나은 세상

- ✅ **사회공헌 카테고리** (환경/기부/봉사/공익) — 카테고리 10번째
- ✅ 사회공헌 챌린지는 둘러보기에 초록 뱃지 + "💚 함께 만든 변화" 배너 + [+ 동참] 버튼
- ✅ 내가 더 나아질수록 세상도 함께 나아짐 — 슬로건의 실제 구현

### 5.8 발견은 사용자 의도로 (v2.4)

- ❌ 알고리즘 추천 (도파민·For You feed) → ✅ **명시 관심 분야 등록** (내정보에 직접 선택)
- 본인이 등록한 카테고리에 해당하는 오픈 도전이 만들어지면 → 홈에 "✨ 관심 도전" 으로 자동 노출
- 시스템은 추천하지 않음 — 사용자가 의도한 만큼만 발견
- "관심 분야 등록" = 본인 정체성 선언 ("나는 운동·사회공헌 사람")

### 5.9 홈 = 도전 인연들의 하루 (v2.5)

- ❌ 챌린지 도구의 "내 streak · 내 진행률" 대시보드 홈 → ✅ **사람들의 되어가는 과정 피드**
- 내 대시보드는 맨 위 me-strip 1줄 ("오늘도 한 걸음 · 오늘 인증 0/2 [인증하기]")
- 본문은 도전 인연들의 하루 — 5종 카드:
  - 🎉 완주 리본 — "지은님이 100일을 완주했어요"
  - 📸 오늘의 인증 — 동료의 사진 + 응원 (좋아요 수 대신 응원한 동료 얼굴)
  - 🙋 응원받기 — "엄마의 100일 다이어트, 오늘도 지켜보는 가족 4명"
  - 🌍 누구나 합류 — 카피·인용 + [함께 합류하기]
  - ✨ 관심 도전 — 본인 명시 관심 매칭 (v2.4)
- 맨 아래 🌙 **"오늘은 여기까지예요. 내일 또, 한 걸음."** — 무한 스크롤 의도적 차단
- ✅ **살린 욕구**: "타인의 되어가는 과정에 닿고 싶다" — 도파민 피드 없이 연결 욕구를 채움

### 5.10 해냈어요 — 자랑 X · 증언 ✓ (v2.5)

- ❌ 완주 일수 랭킹·자랑 톤 → ✅ "줄세우지 않고, 서로에게 용기를"
- 박제 탭의 "완주 이야기 공유" 버튼 → 작성 화면:
  - 시스템 통계 (기간·연속·인증 횟수·완주율) **자동 잠금** — 조작 불가
  - 사진·소감·"가장 어려웠던 점" 옵션
  - 공개 범위: public / 도전 인연에게만
- 공개 탭 카드: 좋아요 수 대신 **"N명에게 용기를 줬어요"**
- 각 이야기 끝에 "나도 비슷한 도전 시작하기" → **신규 유입 루프** (바이럴 엔진)
- ✅ **살린 욕구**: 인정받기 (자아실현 정점) + 타인에게 용기를 전수하는 의미 — 자랑이 아니라 증언

## 6. 핵심 기능 (베타 30명 출시 기준)

### 하단 탭 5개 (v2.5)

| 탭 | 정체성 |
|:-:|---|
| 🏠 **홈** | 도전 인연들의 하루 피드 (SNS-first) |
| 🚩 **내도전** | 내가 참여 중인 방 목록 + 진행률 + 오늘 상태 |
| ⊕ | 새 챌린지 만들기 (5단계 마법사) |
| 🎬 **기록** | 앱 전체 기록(Vlog) 피드 — 영상 중심 |
| 🏆 **해냈어요** | 완주 이야기 공개 탭 ("줄세우지 않고, 서로에게 용기를") |
| 🙂 우상단 아바타 | MY (내정보 · 관심 분야 · 알림 prefs) |

매일 액션 (오늘 인증하기) 은 홈 me-strip 의 [인증하기] 빠른 액션 (오늘 인증 안 한 챌린지 자동 라우팅).

### 챌린지 방 — 4종류

| 이모지 | 종류 | 도전 | 응원 | 둘러보기 |
|:-:|---|---|---|:-:|
| 🤫 | 혼자만의 다짐 | 본인 | 본인 | X |
| 🙋 | **응원받기** | 본인 1명 | 초대된 지인 N명 | X |
| 🤝 | 함께 도전하기 | 멤버 다수 | 멤버 다수 | X |
| 🌍 | 누구나 합류 | 멤버 다수 | 멤버 다수 | O |

> **응원받기 방이 가장 본질적.** "100일 다이어트 선언, 응원해줘" — 친구 신청 없이 지인들이 응원만 해주는 모델.

### 챌린지방 5탭

- 💬 **대화** — Realtime 카톡식 채팅 (멤버 전용)
- 📸 **인증** — 매일 1회 사진 + 4가지 응원 🔥👏💪❤️ + 🎁 선물 (Phase 2)
- 🎥 **기록** — Vlog 형태 글 + 좋아요 (인증과 별개의 추억성 콘텐츠)
- 📊 **현황** — 멤버별 인증률 + 연속 일수 (가입 순, 비교 압박 X)
- 🏆 **박제** — 완주 후 트로피 + 인증 타임라인 + 슬로건

### 만들기 — 5단계 마법사

1. 제목 (인기 추천 칩 6개)
2. 카테고리 — 10 대분류 + 소분류 (건강/운동/학습/창작/자기계발/재테크/라이프/관계/**사회공헌**/기타)
3. 방 타입 — 4종 (혼자 / 응원받기 / 함께 / 누구나)
4. 기간 + 빈도 — 1일~1년 (100일 ⭐) + 매일 / 주 3회 / 주 1회 (1일은 날짜 예약 달력)
5. 인증 방식 — 사진 / 앱 스크린샷 (GPS Phase 2) + 내기는 Phase 2 한 줄 티저

### 둘러보기

- 큐레이션 "지금 핫한" + 카테고리 필터
- 카드: 뱃지 + D-N + 진행률 바 + 4가지 평가 chips + @크리에이터
- 사회공헌 카드 전용: 💚 "함께 만든 변화" 배너 + 초록 컬러 + "+ 동참"

## 7. 베타 30명에게 우리가 약속하는 것

> "30명 중 10명이 친구에게 추천하고 싶다 — 이 응답이 베타의 합격선이에요.
> 안 된다면 무엇이 부족했는지 인터뷰로 듣고, 다음 단계를 정합니다."

### 베타에서 버리는 것

- **무한 스크롤 없음** — 끝이 있는 도전.
- **친구 신청·수락 없음** — 동료는 도전이 만든다.
- **알림 폭격 없음** — 묶음·조용시간·상한.
- **큰 숫자 비교 없음** — 99+ 통일.
- **광고 협찬·내기·박제 결제** = Phase 2. 베타는 순수 검증.

### 베타에서 살리는 것 (v2.5)

- **목격받기** — 동료가 내 인증을 본다 (좋아요 수 X · 응원한 동료 얼굴 ✓)
- **소속** — 같은 목적을 향하는 도전 인연
- **연결** — 도전 인연들의 되어가는 과정 피드 (도파미 X · 의미 ✓)
- **증언** — "해냈어요" 탭. 완주 이야기로 다음 사람에게 용기를 전수

## 8. 베타 사용자 인터뷰 (1주일 후)

### 핵심 질문 (5개)

1. **"앱을 켰을 때, 인스타·카톡 켰을 때와 무엇이 가장 달랐나요?"**
   → 정체성 검증
2. **"매일 인증을 1주일 동안 이어가는 동안, 부담이 줄었나요 늘었나요?"**
   → 조용 알림 / 강제 휴식 / "내일 또 만나요" 정체성 검증
3. **"동료의 응원·댓글·평가를 받았을 때 기분이 어땠나요? 인스타 좋아요와 비교해서?"**
   → 4가지 응원 의미 분화 검증
4. **"이 앱을 친구·가족·동료에게 추천하고 싶다면, 어떤 말로 추천하실 건가요?"**
   → 추천 의향 + 자연 카피 발견
5. **"가장 불편했던 것 1가지 / 가장 좋았던 것 1가지."**
   → 우선순위 발견

### 측정

- 30명 중 1주일 이상 인증 지속: 목표 ≥ 30명
- 챌린지당 평균 멤버: 목표 ≥ 3명
