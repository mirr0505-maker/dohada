베타 앱 설치 가이드

더 나은 나, 더 나은 세상을 향한 첫걸음을 축하합니다!

🎉 베타 테스터 신청이 완료되었습니다!
🍏 iOS 사용자 설치 방법
1 Safari 브라우저로 아래 설치 버튼을 열어주세요.
2 기기에 TestFlight 앱이 없다면 App Store에서 먼저 설치해 주세요.

※ Apple 공식 무료 베타테스트 연동 앱입니다.

3 설치 후 다시 아래의 설치 링크를 열어 [수락(Accept)] 및 [설치(Install)]를 누르면 끝!
👉 TestFlight로 Do:하다 설치하기
🤖 Android 사용자 설치 방법
1 아래의 [베타 설치 페이지 이동] 버튼을 눌러주세요.
2 페이지에서 [Install] 또는 [Download] 버튼을 눌러 .apk 파일을 다운로드합니다.

※ PC로 보시는 경우, 화면의 QR 코드를 휴대폰 카메라로 인식하면 즉시 설치 페이지로 이동합니다.

3 다운로드된 파일을 눌러 설치를 진행합니다. "안전하지 않은 앱" 경고가 뜨면 [무시하고 설치]를 눌러주세요.
4 "Google Play 프로텍트 — 앱 차단됨" 팝업이 뜨는 경우:

① 팝업의 [세부정보 더보기]를 눌러 [무시하고 설치] 항목이 보이면 선택 → 설치 완료!

② [확인] 버튼만 있다면: Play 스토어 앱 → 우상단 프로필 → Play 프로텍트 → ⚙️(설정) → "Play 프로텍트로 앱 검사" 끄기 → APK 파일을 다시 눌러 설치 → 설치 완료 후 검사 옵션을 다시 켜기

👉 Android 베타 파일(.apk) 바로 받기
⚠️ 보안 경고 팝업 안내
최초 설치 시 "출처를 알 수 없는 앱", "안전하지 않은 앱" 또는 "Google Play 프로텍트 차단" 경고가 뜰 수 있습니다. 이는 악성 판정이 아니라, 구글 플레이스토어를 거치지 않고 베타 빌드(.apk)를 직접 설치할 때 표시되는 안드로이드의 일반적인 보안 알림입니다 (Play 스토어에 아직 등록되지 않은 베타 단계라서 "검사한 적이 없다"고 나오는 것입니다). 위 3·4번 절차대로 진행하시면 안전하게 설치되며, 정식 출시 후에는 플레이 스토어에서 경고 없이 받게 됩니다.
🎯 로그인 및 첫 도전 시작 가이드
  • 간편 가입 & 로그인: 앱 실행 후 Google(Android) 또는 Apple/Google(iOS) 로그인으로 3초 만에 시작합니다.
  • 첫 도전 시작: 하단 탭의 [ + (만들기) ] 버튼을 눌러 나만의 첫 도전(1일 ~ 365일 중 선택)을 개설해 보세요!
  • 소중한 피드백 인터뷰: 가입 후 약 1주일 뒤, 이메일로 10분 간단 비대면 인터뷰(소정의 커피 기프티콘 증정) 일정을 발송해 드립니다.
📢 최근 주요 개선 및 패치노트 (v1.0.0 · 최신)
  • 🔭 하다 구경 (NEW): 다른 사람들이 무슨 하다를 하는지 익명으로 구경하고, 마음에 들면 "따라하기"로 바로 내 도전으로 가져와요.
  • 💛 다짐 (NEW): 돈 안 거는 가벼운 약속 — "해내면 ○○ / 못 하면 ○○". 내 정보의 '💛 다짐 내역'에서 진행 중·완주별로 모아볼 수 있어요.
  • 📒 내역 모아보기 (NEW): 내 정보에서 다짐 내역과 ☕ 한잔 내역(건넨/받은 섹션으로 정리)을 한눈에 확인해요.
  • 🛡️ 신고·차단·계정 삭제 (NEW): 부적절한 콘텐츠는 신고, 불편한 사용자는 차단할 수 있고, 계정 삭제도 앱 안에서 바로 돼요.
  • 🤝 응원받기 ↔ 다함께 구분: 내가 '응원하는 방'인지 '직접 도전하는 방'인지 화면이 또렷해졌어요.
  • 🏅 연속 메달 · 📸 사진 여러 장 · 🎯 목표 횟수형 도전: 연속 인증 메달, 인증 3장·기록 4장(좌우 넘김·핀치 줌), "기간 내 N개 달성"형 도전도 그대로 사용 가능해요.
  • ☕ 응원 한잔: 동료 인증에 "한잔"으로 응원 — 베타는 모의 결제라 실제 결제·계좌 연결이 없어요.
  • ✨ 자잘한 개선: 하다 구경 분류 글자 잘림, 끝낸 하다 진행바 색 등 표현·버그를 다듬었어요.
